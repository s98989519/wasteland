# 敵人紀錄 (Enemies)

本文件紀錄遊戲中所有敵人的詳細數值、AI 邏輯與特殊機制。

## 通用機制：護甲 (Armor)
- 護甲可以抵消受到的傷害，比例為 **1 點護甲抵消 1 點傷害**。
- 當敵人受到攻擊時，會優先扣除護甲值，剩餘的傷害才會扣除生命值 (HP)。
- 護甲可以**無限疊加**，不會隨回合消失，直到被玩家的攻擊消耗殆盡為止。

## 野生豬 (Wild Boar) 🐗

### 基礎數值
- **生命值 (HP)**：20 ~ 30
- **基礎攻擊力 (ATK)**：3 ~ 5
- **群體出現機率**：單隻 (25%) / 兩隻 (45%) / 三隻 (20%) / 四隻 (10%)
- **擊殺掉落物** (每隻野生豬獨立計算，戰鬥勝利後統一發放)：
  - **毛皮 (Pelt)**：0~2 個 (20% 掉 0 個，50% 掉 1 個，30% 掉 2 個)
  - **鬃毛 (Mane)**：0~2 個 (20% 掉 0 個，50% 掉 1 個，30% 掉 2 個)
  - **生豬肉 (Pork)**：0~1 個 (60% 掉 0 個，40% 掉 1 個)
  - **獠牙 (Tusk)**：0~1 個 (95% 掉 0 個，5% 掉 1 個)

### AI 狀態機與行動模式
野生豬擁有一套完整的節奏型 AI，包含三種狀態：`normal` (一般)、`preparing` (準備)、`stunned` (暈眩)。

#### 1. 一般狀態 (`normal`)
當回合開始時處於此狀態，會根據以下機率隨機選擇動作：
- **攻擊 (70%)**：
  - 效果：對玩家發起普通攻擊，造成 3~5 點傷害 (受玩家防禦力減免)。
- **裹泥巴 (15%)**：
  - 效果：在泥巴中打滾，為自己增加 **5 點護甲**。
  - 意圖顯示：「裹上泥巴」(橘字)
- **準備 (10%)**：
  - 效果：前腳刨地，不造成傷害。自身狀態切換為 `preparing`。
  - 傷害：0
  - 意圖顯示：「準備衝撞」

#### 2. 準備狀態 (`preparing`)
若上回合使用了「準備」，本回合**必定且唯一**發動以下動作：
- **衝撞 (Charge)**：
  - 效果：發起猛烈衝撞，造成基礎攻擊力 **1.5 倍**的傷害 (約 4~7 點傷害)。攻擊結束後，自身狀態切換為 `stunned`。
  - 意圖顯示：<span style="color:red">衝撞 (預計 X 傷害)</span>

#### 3. 暈眩狀態 (`stunned`)
若上回合發動了「衝撞」，本回合**必定且唯一**發動以下動作：
- **暈眩中 (Stunned)**：
  - 效果：因為猛烈衝撞的後座力而暈眩，無法行動一回合。回合結束後，狀態恢復為 `normal`。
  - 傷害：0
  - 意圖顯示：「暈眩中」

## 綠色巨菇 (Green Mushroom) 🍄

### 基礎數值
- **生命值 (HP)**：16 ~ 26
- **基礎攻擊力 (ATK)**：3 ~ 6
- **群體出現機率**：單隻 (25%) / 兩隻 (45%) / 三隻 (20%) / 四隻 (10%)
- **擊殺掉落物**：
  - **奇異孢子**：0~2 個 (30% 掉 0 個，50% 掉 1 個，20% 掉 2 個)
  - **綠色菇傘**：0~1 個 (75% 掉 0 個，25% 掉 1 個)

### AI 狀態機與行動模式
每回合根據以下機率隨機選擇動作：
- **攻擊 (60%)**：
  - 效果：對玩家發起普通攻擊。
- **治療孢子 (25%)**：
  - 效果：散發出治療孢子，為所有存活的友方單位回復 **5 點生命**。
  - 意圖顯示：「治療孢子」(綠字)
- **縮地 (15%)**：
  - 效果：獲得「閃避」狀態，**完美閃避下一次受到的任何傷害**。
  - 意圖顯示：「縮地 (閃避)」(橘字)

### 被動機制 (死亡觸發)
- **孢子爆裂 (100%)**：
  - 效果：死亡時必定散發出最後的治療孢子，為所有**存活的友方單位恢復 8 點生命**。

## 紅色巨菇 (Red Mushroom) 🍄

- **基礎攻擊力 (ATK)**：4 ~ 7
- **群體出現機率**：單隻 (25%) / 兩隻 (45%) / 三隻 (20%) / 四隻 (10%)

  - **奇異孢子**：0~2 個 (30% 掉 0 個，50% 掉 1 個，20% 掉 2 個)
  - **紅色菇傘**：0~1 個 (75% 掉 0 個，25% 掉 1 個)

### AI 狀態機與行動模式
每回合根據以下機率隨機選擇動作：
- **攻擊 (70%)**：
  - 效果：對玩家發起普通攻擊。
- **狂暴孢子 (20%)**：
  - 效果：散發出狂暴孢子，使所有存活的友方單位**攻擊力永久提升 2 點** (可無限疊加)。
  - 意圖顯示：「狂暴孢子」(紅字)
- **縮地 (10%)**：
  - 效果：獲得「閃避」狀態，**完美閃避下一次受到的任何傷害**。
  - 意圖顯示：「縮地 (閃避)」(橘字)

### 被動機制 (死亡觸發)
- **孢子爆裂 (100%)**：
  - 效果：死亡時必定噴發出最後的狂暴孢子，為所有**存活的友方單位附加 1 層狂暴 (+2 ATK)**。
